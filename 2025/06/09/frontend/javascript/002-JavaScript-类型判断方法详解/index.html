4. typeof null 的特殊性

原因：
JavaScript 早期设计时用二进制前三位标识类型：

  • 000：对象类型
  • 1：整型
  • 010：双精度浮点型
  • 100：字符串
  • 110：布尔型
    null 的二进制全为 0，被错误识别为对象类型

5. typeof NaN 的特殊性

NaN 的特性：

  • 全称：Not-A-Number（非数字）
  • 是特殊的 Number 类型值
  • 由无效数学运算产生：0/0, Math.sqrt(-1)
  • 不等于任何值（包括自身）：NaN === NaN → false
